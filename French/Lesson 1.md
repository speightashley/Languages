# Perception Drill

1. Est-ce que le taxi est là?
2. Oui, il est là
3. Où est le sac?
4. Ici.
5. Où est Mme Capet?
6. Elle est ici.



>[!Note]
> Est-ce que + statement -> Question Form

Est-ce que le taxi est là?
- Oui, il est là

Est-ce que M. Capet est là?
- Oui, il est là.

Est-ce qu'il est là?
- Oui, il est là.
